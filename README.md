## 📖 项目简介

终末地基建模拟器是一个功能强大的网页应用，帮助玩家规划和优化《明日方舟：终末地》中的基地布局、生产链和资源配置。

支持两大地区（四号谷地、武陵）共 6 个不同区域的完整规划，提供可视化的基地设计、生产计算和蓝图管理功能。

## ✨ 主要功能

### 🎨 可视化画布
- 拖拽式基地布局设计
- 支持多层建筑规划（传送带层/管道层）
- 可调节网格大小、透明度和显示/隐藏
- 物品的放置、旋转、删除和框选操作

### �️ 地区系统
- **四号谷地**: 枢纽区、谷地通道、矿石研究园、供能高地
- **武陵**: 武陵城、景玉谷
- 每个地区拥有独特的网格配置和布局

### 📦 物品与配方
- 完整的物品数据库
- 详细的生产配方信息
- 快速搜索和筛选功能

### 🧮 生产计算器
- 多目标物品同时计算
- 自动计算完整生产链
- 显示原材料需求和中间产物
- 设备数量建议和自动布局

### 📐 蓝图系统
- 保存和加载基地布局
- 导出/导入蓝图数据
- 蓝图预览和统计信息

### 🔧 物流规划
- 传送带和管道的智能路径规划
- 可视化物流路径设计
- 分流器和汇流器配置

### ⚡ 电力统计
- 实时监控基地功耗
- 供电桩和中继站统计
- 耗电设备排行

### 🌐 国际化
- 完整的中文/英文翻译
- 所有物品名称本地化
- 地区名称多语言支持
- 实时语言切换
